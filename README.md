# 🚀 FlySend - Peer-to-Peer Package Delivery

FlySend connects people who want to send packages to Ethiopia with travelers who have spare luggage space. Send berbere, coffee, electronics, and gifts at a fraction of traditional shipping costs!

## ✨ Features

- **User Authentication** - Secure signup/login with Firebase
- **Listing System** - Post items to send or travel plans
- **Smart Matching** - Automatic connection between senders and travelers
- **Real-time Chat** - Secure in-app messaging
- **Trust System** - User ratings and reviews
- **Beautiful UI** - Modern, responsive design

## 🛠️ Tech Stack

- **Frontend**: Next.js 14 + TypeScript + Tailwind CSS
- **Backend**: Firebase (Auth, Firestore, Storage)
- **Hosting**: Vercel
- **Payments**: Stripe (coming soon)
